# A quiet place to think out loud

*June 23, 2026*

This is a log. Not quite a blog — there's no banner announcing it, no link in
the nav. You found it by hovering the right four words in a footer, or by
guessing the URL. Either way: welcome.

I'll keep notes here. Things I'm building, things I'm reading, half-formed
thoughts I'd rather not post anywhere louder.

To add a new entry:

1. Drop a Markdown file in `/log/` — say `something.md`.
2. Add one line to `log.html` pointing at it:
   `<a href="/log/post.html?p=something" data-date="2026-07-01">Title</a>`

The list sorts itself newest-first. That's the whole system.
